# Chainlit Train
Este repositório é uma prova de conceito (POC) do uso da ferramenta ChainLit com modelos de LLM.

Para esta abordagem, é utilizado modelos do Ollama rodando localmente.

## Rodando o projeto


### Ollama
Primeiro, se for manter modelos de LLM localmente, certifique-se de instalar o Ollama e baixar o modelo desejado.

### Dependências
```
pip install -r requirements.txt
```

### Executando
```
chainlit run demo.py -w
```
